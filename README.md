
# KeyLogger 

## About the Project
This is a simple C# Console Application. I created this project to understand how Windows APIs work and how the operating system handles keyboard events using `User32.dll`. 

## Built With
* C# 
* .NET Framework
* Visual Studio


## How to Run
1. Clone the repository to your local machine.
2. Open the project in Visual Studio.
3. Build and run the Console Application.
